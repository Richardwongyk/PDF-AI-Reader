# PDF AI Reader

> Attention is all you need.  
> 但读论文时，你还需要翻译、提问、公式、证据和一个不打断思路的阅读空间。

PDF AI Reader 是一个面向学术论文的智能 PDF 阅读器。它希望把论文从一份静态文件，变成一个可以边读、边问、边理解的工作台。

你可以像普通 PDF 阅读器一样打开论文、滚动、缩放和跳转；也可以直接双击段落翻译，围绕全文提问，让 AI 帮你解释概念、梳理公式和定位依据。

---

## 它能做什么

### 流畅读论文

打开 PDF 后直接进入阅读状态。页面、目录、缩放、滚动和段落交互都围绕长篇论文阅读设计，重点是让你尽快回到正文，而不是先处理工具本身。
当前界面默认使用黑色背景阅读风格；菜单栏“主题”和设置窗口可在深色、米黄、浅灰、黑白四套主题之间切换。深色主题背景保持既有黑底契约；米黄、浅灰和黑白主题参考远程版本配色，并在侧栏、输入框、表格和右侧回答区保持足够对比度。翻译框与批注框继续使用原有蓝色和护眼色，不随主题改色。

应用会把最后一次阅读的 PDF、文档 hash、页号和页内滚动位置保存到本地 `data/reader_session.json`。下次普通启动时，如果文件仍存在且内容 hash 未变化，会自动重新打开并回到上次阅读位置；如果文件被删除或内容被替换，则跳过位置恢复，不影响正常打开。

### 双击就翻译

遇到难读的段落，双击即可翻译。翻译会尽量保留论文里的公式、符号和上下文，让你不必在阅读器、翻译网页和笔记软件之间来回切换。

### 对整篇论文提问

你可以问：

- 这篇论文解决了什么问题？
- Self-Attention 为什么能替代循环结构？
- 这个公式里的每个符号是什么意思？
- 作者在哪一页解释了实验设置？

回答会尽量基于当前 PDF 的内容，而不是泛泛聊天。

### 带着证据回答

论文阅读最怕"看起来很对"。PDF AI Reader 的目标是让回答能回到原文：页码、段落、公式和上下文都应该可以追溯，方便你确认 AI 没有偏离论文。

### 更认真地处理公式

很多论文的关键都藏在公式里。项目正在持续增强公式识别、公式审核和公式知识库能力，让数学内容不只是被当成普通文本处理。

### 为深度阅读而生

它适合阅读机器学习、数学、物理、计算机系统等公式密集、上下文复杂的论文。无论是快速扫读一篇 paper，还是反复研究一篇经典论文，都可以把阅读、翻译、问答和证据放在同一个界面里。

## 使用指南

### 安装与启动

1. **安装依赖**（需要 Python 3.14.4）：

   ```powershell
   # 创建 conda 环境并安装依赖
   conda create -n pdf_ai_reader_314 python=3.14.4
   conda activate pdf_ai_reader_314
   pip install -r requirements.txt
   ```

2. **配置模型**：复制 `config.example.yaml` 为 `config.yaml`，填写你的 API Key 和模型偏好：

   ```yaml
   model:
     local: "qwen3.5:4b"          # 本地 Ollama 模型（翻译/问答）
     cloud: "deepseek/deepseek-v4-flash"  # 云端模型（兜底/精扫）
     embedding: "bge-m3"           # 嵌入模型（知识库检索）
     ollama_host: "http://localhost:11434"
   api_keys:
     deepseek-v4-flash: "sk-your-key-here"
   ```

3. **启动应用**：

   ```powershell
   python src\main.py
   ```
   或双击 `run_py314.vbs` 静默启动（无命令行窗口），`run_py314.bat` 可见日志。

> 项目附带 `《Attention Is All You Need》.pdf` 用于演示，启动后可直接打开体验翻译和问答功能。

### 基本操作

| 操作 | 方式 |
|------|------|
| 打开 PDF | 菜单栏 **文件 → 打开 PDF**，或点击工具栏 📂 按钮 |
| 翻页/滚动 | 鼠标滚轮 或 触控板双指滑动 |
| 缩放 | `Ctrl+=` 放大，`Ctrl+-` 缩小 |
| 页码跳转 | 工具栏输入页码后按 Enter |
| 目录跳转 | 左侧导航栏点击章节标题 |
| 隐藏工具栏 | 双击工具栏空白区域或菜单栏文字区域 |
| 显示工具栏 | 双击顶部蓝色细条 |

### 翻译段落

**双击 PDF 中的任意段落**，该段落下方会"裂开"显示翻译结果。

- 翻译会自动保留公式、符号和上下文。
- 再次双击可折叠/展开翻译。
- 点击 **⟳ 重新翻译** 可绕过缓存重新请求。

### 全文问答

在右侧 **AI 工具集**面板中输入问题，AI 会检索全文知识库后回答：

- 问题示例：
  - "这篇论文解决了什么问题？"
  - "作者在哪一页解释了实验设置？"
  - "这个公式里的每个符号是什么意思？"

- **证据面板**：检索到的原文片段会列出页码和相关度评分。
- **双击证据项**可跳转到原文对应位置。
- **追问按钮**：回答完成后自动生成建议问题，一键追问。

### 概念解释

**右键段落 → 请解释**，AI 会结合上下文解释该段落中的核心概念。

### 公式处理

- **自动识别**：打开 PDF 后，应用会自动检测并识别文档中的公式。
- **公式精扫**：点击工具栏 **公式精扫** 按钮，对当前视口公式进行高精度 OCR。
- **公式审核**：菜单栏 **工具 → 公式候选审核**，可逐个审核识别结果。

### 主题切换

菜单栏 **视图 → 主题** 即可切换，自动保存到 `config.yaml`：

| 选项 | 风格 | 强调色 |
|------|------|--------|
| `light` 米黄 | 暖米黄底 + 深灰蓝字 | 琥珀 `#d4902a` |
| `dark` 浅灰 | 中灰底 + 深灰字 | 浅蓝 `#5b9bd5` |
| `sepia` 黑白 | 黑底 + 白字 | 靛紫 `#6c5ce7` |

也可直接在 `config.yaml` 中修改 `ui.theme`。

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 打开 PDF |
| `Ctrl+=` / `Ctrl+-` | 放大 / 缩小 |
| `Ctrl+Enter` | 发送提问（在输入框中） |
| `Esc` | 折叠当前展开的翻译/问答 |
| 双击段落 | 翻译 / 折叠切换 |
| 双击工具栏空白 | 隐藏工具栏 |
